# AI Masterclass landing — Voltage

Self-contained static landing page (design variant **08-voltage**) for Azizbek Vafoyev's
free 2-day AI master-class. No build step, no dependencies — just `index.html`, the photo
in `assets/`, Google Fonts (CDN), and a tiny inline countdown script.

## Deploy

- **Vercel:** drag this folder into the Vercel dashboard, or run `vercel --prod` inside it.
  No framework / build settings needed (static).
- **Local preview:** `python3 -m http.server` then open http://localhost:8000

The countdown targets **2026-07-02 20:30 (UTC+5)**; both CTA buttons link to the Telegram group.
